# Alibaba Translation Tool

## Information
This project is a tool for translating text using Alibaba Cloud Translate API.just a simple demo to translate from Chinese to English .
for more information please visit [https://help.aliyun.com/zh/machine-translation/developer-reference/](https://help.aliyun.com/zh/machine-translation/developer-reference/)

## Requirements
- Java Development Kit (JDK) 8 or higher
- Internet connection for cloud-based services

## Usage

1. Clone the repository
2. Replace `ALI_SECRET` and `ALI_ACCESS_KEY` to your own aliyun access key and secret.
